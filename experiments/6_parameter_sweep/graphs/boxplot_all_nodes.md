# Box Plot: All Nodes Parameter Sensitivity

## Description

Companion to the violin plot, this box plot shows latency distribution for all 15 nodes using standard statistical quartile representation.

## How to Read

- **Box**: Interquartile range (IQR, 25th to 75th percentile).
- **Notch**: 95% confidence interval for the median.
- **Red line**: Median latency across all parameter configurations.
- **Whiskers**: Extend to 1.5x IQR.
- **Diamond markers**: Outlier parameter configurations that produce unusually high or low latencies.
- Nodes are ordered by median latency (highest to lowest).
