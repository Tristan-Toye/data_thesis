# Node Latency Table (HTML)

**File:** `node_latency_table.html`

Interactive HTML table of all profiled nodes with colour-coded priority:

| Colour | Meaning |
|---|---|
| Red | >10% of total latency -- high priority |
| Yellow | 5--10% -- medium priority |
| Green | <5% -- low priority |

Open the file in a browser to sort columns, hover for details, and quickly identify which nodes dominate system latency.
